# learn-project

למידה מעמיקה של הפרויקט - חקירה דינמית של הקוד הנוכחי.

## Instructions

כשמפעילים את הסקיל הזה, בצע חקירה מעמיקה של הפרויקט:

### 1. קרא את הקבצים המרכזיים
קרא את הקבצים הבאים בפועל כדי להבין את המצב הנוכחי:

**ליבה:**
- `core/logic.py` - הAPI הראשי לחישוב
- `core/wage_calculator.py` - מנוע חישוב השכר
- `core/segments.py` - עיבוד סגמנטים
- `core/constants.py` - קבועים

**תצוגה:**
- `app_utils.py` - נתוני סגמנטים יומיים
- `routes/guide.py` - תצוגה מפורטת

**הגדרות:**
- `core/config.py` - קונפיגורציה

### 2. בדוק שינויים אחרונים
```bash
git log --oneline -10
git diff --stat HEAD~5
```

### 3. הבן קשרים
- איך `logic.py` ו-`app_utils.py` מתקשרים
- איך הסגמנטים מחושבים
- איך הרצפים (chains) עובדים

### 4. דווח למשתמש
לאחר הקריאה, תן סיכום קצר של:
- מה למדת
- שינויים אחרונים שראית
- נקודות חשובות לזכור

### 5. היה מוכן לשאלות
לאחר הלמידה, היה מוכן לענות על שאלות ספציפיות על הקוד.

## דגשים
- **תמיד קרא קבצים בפועל** - אל תסתמך רק על CLAUDE.md
- **שים לב לדואליזם** - יש חישוב ב-logic.py וגם ב-app_utils.py
- **בדוק שינויים אחרונים** - הקוד משתנה
